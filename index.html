Penjelasan Program Python - Kivy

Tujuan: Menampilkan tombol di tengah layar menggunakan GUI framework Kivy.

Penjelasan Baris per Baris:

  • from kivy.app import App: Mengimpor kelas utama untuk membuat aplikasi Kivy.
  • from kivy.uix.button import Button: Untuk membuat tombol.
  • from kivy.uix.floatlayout import FloatLayout: Layout fleksibel untuk atur posisi elemen.
  • from kivy.core.window import Window: Untuk mengatur warna latar aplikasi.
  • from kivy.utils import get_color_from_hex: Untuk mengubah warna dari format hex ke Kivy.
  • Window.clearcolor = ...: Mengatur latar belakang aplikasi menjadi warna abu-abu muda.
  • btn1 = Button(...): Membuat tombol dengan ukuran dan posisi tertentu.
  • btn1.background_color = ...: Mengubah warna tombol.
  • f_layout.add_widget(btn1): Menambahkan tombol ke dalam layout.
  • return f_layout: Menampilkan layout sebagai tampilan utama aplikasi.
  • MyApp().run(): Menjalankan aplikasi Kivy.
