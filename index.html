Penjelasan Program Python Kivy

Tujuan Program: Menampilkan tombol di tengah layar menggunakan GUI Framework Kivy, dengan pengaturan warna latar belakang dan layout yang fleksibel.

Penjelasan Baris per Baris Kode:

  • from kivy.app import App: Mengimpor kelas dasar App yang digunakan untuk membuat dan menjalankan aplikasi Kivy.
  • from kivy.uix.button import Button: Mengimpor komponen Button untuk membuat tombol dalam aplikasi.
  • from kivy.uix.floatlayout import FloatLayout: Mengimpor FloatLayout, yaitu layout fleksibel yang memungkinkan penempatan widget secara bebas menggunakan pos_hint.
  • from kivy.core.window import Window: Mengimpor objek Window untuk mengatur tampilan jendela aplikasi seperti warna latar belakang.
  • from kivy.utils import get_color_from_hex: Mengimpor fungsi untuk mengubah kode warna hex (misalnya #eeeeee) menjadi format warna Kivy (RGBA).
  • Window.clearcolor = get_color_from_hex('#eeeeee'): Mengatur warna latar belakang jendela aplikasi menjadi abu-abu muda.
  • btn1 = Button(text='btn1', size_hint=(0.2, 0.2), pos_hint={'x': 0.4, 'y': 0.4}): Membuat tombol dengan teks "btn1", ukuran 20% dari layar, dan diletakkan di tengah menggunakan pos_hint.
  • btn1.background_color = get_color_from_hex('#aaaaaa'): Mengubah warna latar belakang tombol menjadi abu-abu gelap.
  • f_layout = FloatLayout(): Membuat objek FloatLayout sebagai layout utama aplikasi.
  • f_layout.add_widget(btn1): Menambahkan tombol btn1 ke dalam layout agar tampil di aplikasi.
  • return f_layout: Mengembalikan layout sebagai tampilan utama aplikasi saat dijalankan.
  • MyApp().run(): Menjalankan aplikasi Kivy yang telah dibuat.
